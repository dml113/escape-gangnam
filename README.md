# 강남 방탈출 예약 조회

강남 방탈출 카페 4곳의 예약 가능 시간을 한 번에 조회하는 스크립트

## 조회 대상

| 카페 | 위치 | 잼핏 평점 | 5인 가격 |
|------|------|---------|---------|
| [제로월드 강남점](https://zerogangnam.com/reservation) | 강남역 | ⭐ 4.4~4.8 | 110,000원 |
| [CODE-K 강남점](http://www.code-k.co.kr) | 강남역 | ⭐ 2.6~3.5 | 90,000원 |
| [미스터리룸 강남점](http://mysteryroomescape-gn.com) | 강남역 | ⭐ 2.4 | 90,000원 |
| [이룸에이트](https://eroom8.co.kr) | 신논현역 | ⭐ 2.8~3.8 | 120,000원 |

## 설치

```bash
pip install requests beautifulsoup4
```

## 사용법

```bash
# 오늘 날짜 조회
python3 escape_gangnam.py

# 특정 날짜 조회
python3 escape_gangnam.py 2026-03-12
```

## 참고

- CODE-K, 이룸에이트는 약 7~8일 이내 날짜만 조회 가능
- 제로월드는 anti-scraping으로 예약 상태가 부정확할 수 있음 → 실제 예약은 각 사이트에서 직접 확인
